<p><ul><li>Shape Memory Polymer (SMP)</li><li>Liquid Crystal Elastomer (LCE)</li><li>Shape Memory Alloy (SMA)</li><li>Hydrogel</li></ul></p>
<p><p>Shape Memory Polymer (SMP), Liquid Crystal Elastomer (LCE), Shape Memory Alloy (SMA), and Hydrogel are all types of materials used in the 4D printing market. SMPs are capable of changing shape in response to stimuli like heat or light. LCEs have the ability to change shape and stiffness when exposed to external factors. SMAs can "remember" their original shape and return to it when triggered. Hydrogels are responsive to changes in pH, temperature, or electrical stimuli, allowing for dynamic shape changes.</p></p>
